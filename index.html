Sentinel is prepared to prevent trouble before it starts.

Features:

Global Ban List

Sentinel features a Global Ban List holding a database of reported users, ranging everything from DM Advertisers to Raiders.

Sentinel Early WarningTM

Sentinel features an early warning system to let you know when a global banned user has entered your server*

Sentinel AutobanTM

Sentinel features an autoban system taking the work out of protecting your server. When a global banned user enters your server, Sentinel will automatically ban the user for you.**

How to Use:

1. Set your log channel.

Set your log channel using s?log <channel_id / channel_mention>


2. Toggle your features.

Toggle your features on and off depending on what you want to use. s?toggle for a list of modules.
e.g s?toggle 1 enable


3. Sit back and let Sentinel go to work

Sit back, relax and let Sentinel do the work for you.

*Requires log channel to be set and logging module to be toggled.

**Requires log channel to be set, autoban module to be toggled and BAN_MEMBERS permission to be granted.
